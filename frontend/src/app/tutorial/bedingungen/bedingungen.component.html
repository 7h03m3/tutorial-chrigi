Bedingungen

Erklärung
Mit Bedingungen können Werte (z.B. Nummern) einer oder mehreren Variablen verglichen werden. Als Resultat kommt ein binärer Wert (true / false) heraus. Dies wird auch ein boolean genannt. Es wird dabei immer von links nach rechts aufgelöst.
Es gibt die folgenden Vergleichsoperatoren:
  • == Gleich
  • != Ungleich
  • > Grösser
  • < Kleiner
  • >= Grösser-Gleich
  • <= Kleiner-Gleich
Mit den folgenden Operatorn können die Bedingungen logisch verknüpft werden:
  • && Und / AND
  • || Oder / OR
  • ! Nicht / NOT
Werden Bedingungen miteinander verknüpft, müssen Klammern (...) verwendet werden. Hier ein Beispiel:
(Zahl1 == 0) || (Zahl1 == 2) Trifft zu wenn Zahl1 gleich 0 oder 2 ist.
number1 number2
